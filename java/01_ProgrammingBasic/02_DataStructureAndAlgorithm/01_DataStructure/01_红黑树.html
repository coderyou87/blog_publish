# 红黑树

红黑树是一种含有红黑节点并能自平衡的二叉搜索树。红黑树需要满足的几个性质：

  • 每个节点要么是黑色的、要么是红色的
  • 根节点是黑色的
  • 每个叶子节点是黑色的（这里的叶子节点是null节点，也就是说叶子节点中没有存储任何信息）
  • 每一个红色节点的两个子节点都一定是黑色的（不能存在两个红色节点相连的情况）
  • 从任意一个节点出发，到其子树的叶子节点，所经过的路径中，包含的黑色节点数量相同。（推论：如果一个节点包含一个黑色子节点，那么该节点一定含有两个子节点）

红黑树能够达到自平衡，主要依靠的是：左旋、右旋和变色

  • 左旋：以某个节点为支点（旋转节点），其右子节点旋转为父节点，其右子节点的左子节点变为旋转节点的右子节点，其余不变

  • 右旋：以某个节点为支点（旋转节点），其左子节点旋转为父节点，其左子节点的右子节点变为旋转节点的左子节点，其余不变

  • 变色：节点的颜色由红变黑或者由黑变红

红黑树的插入：

  1. 找到插入的位置
  2. 插入后调整节点位置，达到自平衡
  3. 新插入的节点是红色的

红黑树插入的情况分析：

  1. 插入节点时，红黑树为空

    因为插入节点时，红黑树为空，所以直接将新插入的节点作为根节点即可。

  2. 待插入的节点key值已经存在

    节点位置与颜色不变，更改节点中存储的数据即可

  3. 待插入节点的父节点是黑色的

    由于新插入的节点是红色的，并且父节点是黑色的，所以插入节点后，并没有破坏红黑树的平衡，所以直接插入值即可。

  4. 插入节点的父节点是红色的

    因为红黑树的根节点是黑色的，所以如果新插入节点的父节点是红色的，那么此父节点一定不是根节点，即新插入的节点总是存在爷爷节点的。这一点是很关键的，因为后续的旋转需要爷爷节点的参与。

    1. 叔叔节点存在并且为红色

      根据红黑树的性质（不能存在两个红色节点相连），当父亲与叔叔节点都为红色的时候，爷爷节点一定是黑色的。又因为新插入的节点是红黑的，所以就形成了LL双红的情况。处理步骤如下：

      • 将父亲与叔叔节点变为黑色
      • 将爷爷节点变为红色
      • 将爷爷节点设置为当前节点，进入下一轮的处理

    2. 叔叔节点不存在，插入节点的父节点是爷爷节点的左子节点

      1. 新插入的节点，为其父节点的左子节点（LL双红）

        • 将父亲节点（P）设置为黑色
        • 将爷爷节点（PP）设置为红色
        • 以爷爷节点（PP）为旋转节点，进行右旋

      2. 新插入的节点，为其父节点的右子节点（LR双红）

        • 对父亲节点（P）进行左旋，形成LL双红的情况
        • 将P节点设置为当前节点
        • 按照LL双红情况处理

    3. 叔叔节点不存在，插入节点的父节点是爷爷节点的右子节点

      1. 新插入的节点，为其父节点的右子节点（RR双红）

        • 将父亲节点（P）设置为黑色，将爷爷节点（PP）设置为红色
        • 以爷爷节点为旋转节点，进行左旋

      2. 新插入的节点，为其父节点的左子节点（RL双红）

        • 以父亲节点为旋转节点，进行右旋
        • 将父亲节点设置为当前节点，形成RR双红的情况
        • 按照RR双红的情况进行处理

    代码：

    package com.it.rbt;

public class RBTree<K extends Comparable<K>, V> {

    private static final Boolean RED = true;
    private static final Boolean BLACk = false;
    private RBNode root;


    /**
     * get parent node
     *
     * @param node
     * @return
     */
    public RBNode getParent(RBNode node) {
        return node == null ? null : node.getParent();
    }

    /**
     * 判断当前节点的颜色是否为红色
     * @param node
     * @return
     */
    public Boolean isRed(RBNode node) {
        return (node != null) && (node.color == RED);
    }

    /**
     * 判断当前节点的颜色是否为黑色
     * @param node
     * @return
     */
    public Boolean isBlack(RBNode node) {
        return (node != null) && (node.color == BLACk);
    }

    public void setRed(RBNode node) {
        if (node != null) {
            node.color = RED;
        }
    }

    public void setBLACk(RBNode node) {
        if (node != null) {
            node.color = BLACk;
        }
    }

    /**
     * 中序遍历红黑树
     */
    public void inOrderPrint() {
        inOrderPrint(this.root);
    }

    public void inOrderPrint(RBNode node) {
        if (node != null) {
            inOrderPrint(node.left);
            System.out.println("key: " + node.key + ", value:  " + node.value + ", color: " + node.color);
            inOrderPrint(node.right);
        }
    }

    /**
     * 左旋
     *      p           p
     *      |           |
     *      x           y
     *     / \          /\
     *    lx  y        x  ry
     *        /\      /\
     *       ly ry   lx ly
     *
     *
     * @param x
     */
    public void leftRotate(RBNode x) {
        RBNode y = x.right;

        // 变更x与ly的关系
        // 将x的right指向y的left
        // 将ly的parent指向x
        x.right = y.left;
        if (y.left != null) {
            y.left.parent = x;
        }

        // 变更p与y的关系
        // 将y的parent指向x的parent
        // 将p的left或right指向y
        if (x.parent != null) {
            y.parent = x.parent;
            if (x == x.parent.left) {
                x.parent.left = y;
            } else {
                x.parent.right = y;
            }
        } else {
            // 说明原来的x是根节点，旋转后y是根节点，并将y的parent指向null
            this.root = y;
            this.root.parent = null;
        }

        // 变更x与y的关系
        // 将x的parent指向y
        // 将y的left指向x
        x.parent = y;
        y.left = x;
    }

    /**
     * 右旋
     *          p           p
     *          |           |
     *          y           x
     *         / \         / \
     *        x   ry      lx  y
     *       / \             / \
     *      lx  rx          rx  ry
     *
     *
     * @param y
     */
    public void rightRotate(RBNode y) {
        RBNode x = y.left;
        // 变更y与rx的关系
        y.left = x.right;
        if (x.right != null) {
            x.right.parent = y;
        }

        // 变更p与x的关系
        if (y.parent != null) {
            x.parent = y.parent;
            if (y == y.parent.left) {
                y.parent.left = x;
            } else {
                y.parent.right = x;
            }
        } else {
            this.root = x;
            this.root.parent = null;
        }

        // 变更x与y的关系
        y.parent = x;
        x.right = y;
    }

    /**
     * 向红黑树中插入新的节点
     * @param key
     * @param value
     */
    public void insert(K key, V value) {
        RBNode node = new RBNode();
        node.setKey(key);
        node.setValue(value);
        node.setColor(RED);
        insert(node);
    }

    public void insert(RBNode node) {
        RBNode parent = null;
        RBNode current = this.root;
        while (current != null) {
            parent = current;
            int compare = node.key.compareTo(current.key);
            // compare > 0 : node.key > current.key, search in right tree
            // compare = 0 : node.key = current.key, replace value
            // compare < 0 node.key < current.key, search in left tree
            if (compare > 0) {
                current = current.right;
            } else if (compare < 0) {
                current = current.left;
            } else {
                current.setValue(node.getValue());
                return;
            }
        }
        node.parent = parent;
        if (parent != null) {
            int compare = node.key.compareTo(parent.key);
            // compare > 0 : node.key > current.key, insert right tree
            // compare = 0
            // compare < 0 node.key < current.key, insert left tree
            if (compare > 0) {
                parent.right = node;
            } else {
                parent.left = node;
            }
        } else {
            this.root = node;
        }
        fixRBTree(node);
    }

    /**
     * 新增节点后，可能会破坏红黑树的平衡，需要修复
     *  |--情景1：红黑树为空，将根节点颜色为黑色
     *  |--情景2：插入节点的key已经存在，不做处理（只是替换节点的值，不涉及节点增删，红黑树依然保持平衡）
     *  |--情景3：插入节点的父节点为黑色，不做处理（因为插入节点后，红黑树依然保持平衡）
     *  |--情景4：插入节点的父节点为红色（需要手动处理）
     *      |--4.1：叔叔节点存在，并且为红色（父亲与叔叔都是红色），将父亲与叔叔染色为黑色，将爷爷染色为红色，并且将爷爷节点设置为当前节点，继续进行下一轮处理
     *      |--4.2：叔叔节点不存在，或者存在且为黑色，插入节点的父节点为爷爷节点的左子树
     *          |--4.2.1：插入节点为其父节点的左子节点（LL），将父亲染色为黑色，将爷爷染色为红色，以爷爷节点为基础，进行右旋
     *          |--4.2.2：插入节点为其父节点的右子节点（LR），以父亲节点为基础，进行左旋，得到LL双红（4.2.1），将父亲节点设置为当前节点，进入下一轮处理
     *      |--4.3：叔叔节点不存在，或者存在且为黑色，插入节点的父节点为爷爷节点的右子树
     *          |--4.3.1：插入节点为其父节点的右子节点（RR），将父亲节点染色为黑色，将爷爷节点染色为红色，以爷爷节点为基础，进行左旋
     *          |--4.3.2：插入节点为其父节点的左子节点（RL），以父亲节点为基础，进行右旋，得到RR双红（4.3.1），将父节点设置为当前节点，进入下一轮处理
     */
    public void fixRBTree(RBNode node) {
        this.root.setColor(BLACk);
        RBNode parent = getParent(node);
        RBNode grandParent = getParent(parent);
        // 4
        //if (parent == null || isBlack(parent)){
        //    return;
        //}

        if (parent != null && isRed(parent)) {
            RBNode uncle = null;
            // 如果父亲节点是红色，那么爷爷节点一定存在，因为根节点不可能是红色
            // 父节点为爷爷节点的左子树
            if (parent == grandParent.left) {
                uncle = grandParent.right;
                // 4.1 叔叔节点存在 并且为红色
                if (uncle != null && isRed(uncle)) {
                    //将父亲与叔叔染色为黑色，将爷爷染色为红色，并且将爷爷节点设置为当前节点，继续进行下一轮处理
                    setBLACk(parent);
                    setBLACk(uncle);
                    setRed(grandParent);
                    fixRBTree(grandParent);
                    return;
                }
                // 4.2 叔叔节点不存在，或者存在且为黑色，插入节点的父节点为爷爷节点的左子树
                if (uncle == null || isBlack(uncle)) {
                    //4.2.1：插入节点为其父节点的左子节点（LL）
                    if (node == parent.left) {
                        //将父亲染色为黑色，将爷爷染色为红色，以爷爷节点为基础，进行右旋
                        setBLACk(parent);
                        setRed(grandParent);
                        rightRotate(grandParent);
                        return;
                    }
                    //4.2.2：插入节点为其父节点的右子节点（LR）
                    if (node == parent.right) {
                        //以父亲节点为基础，进行左旋，得到LL双红（4.2.1），将父亲节点设置为当前节点，进入下一轮处理
                        leftRotate(parent);
                        fixRBTree(parent);
                    }
                }

            } else {
                // 父节点为爷爷节点的右子树
                uncle = grandParent.left;
                // 4.1 叔叔节点存在 并且为红色
                if (uncle != null && isRed(uncle)) {
                    //将父亲与叔叔染色为黑色，将爷爷染色为红色，并且将爷爷节点设置为当前节点，继续进行下一轮处理
                    setBLACk(parent);
                    setBLACk(uncle);
                    setRed(grandParent);
                    fixRBTree(grandParent);
                    return;
                }

                // 4.3 叔叔节点不存在，或者存在且为黑色，插入节点的父节点为爷爷节点的右子树
                if (uncle == null || isBlack(uncle)) {
                    //4.3.1：插入节点为其父节点的右子节点（RR）
                    if (node == parent.right) {
                        //将父亲节点染色为黑色，将爷爷节点染色为红色，以爷爷节点为基础，进行左旋
                        setBLACk(parent);
                        setRed(grandParent);
                        leftRotate(grandParent);
                    }
                    //4.3.2：插入节点为其父节点的左子节点（RL）
                    if (node == parent.left) {
                        //以父亲节点为基础，进行右旋，得到RR双红（4.3.1），将父节点设置为当前节点，进入下一轮处理
                        rightRotate(parent);
                        fixRBTree(parent);
                    }
                }
            }
        }
    }


    static class RBNode<K extends Comparable<K>, V> {
        private RBNode parent;
        private RBNode left;
        private RBNode right;
        private boolean color;
        private K key;
        private V value;

        public RBNode() {
        }

        public RBNode(RBNode parent, RBNode left, RBNode right, boolean color, K key, V value) {
            this.parent = parent;
            this.left = left;
            this.right = right;
            this.color = color;
            this.key = key;
            this.value = value;
        }
		// setter、getter
    }


    public static void main(String[] args) {

        RBTree<Integer, Object> rbTree = new RBTree();
        rbTree.insert(6, null);
        rbTree.insert(22, null);
        rbTree.insert(27, null);
        rbTree.insert(1, null);
        rbTree.insert(11, null);
        rbTree.insert(15, null);
        rbTree.insert(25, null);
        rbTree.insert(8, null);
        rbTree.insert(17, null);
        rbTree.insert(13, null);
        rbTree.inOrderPrint();
    }
}
