# LeetCode72 编辑距离

分析：

  • 状态表示f[i,j]：
    • 集合：所有将第一个字符串的前i个字母变为第二个字符串的前j个字母的方案
    • 属性：集合中，所有方案的最小值（将第一个字符串的前i个字母变为第二个字符串的前j和字母，有多种方案，每种方案的操作数可能不同）
  • 状态计算：将第一个字符串的前i个字母变为第二个字符串的前j个字母，最后一步操作有三种情况
    • insert：如果最后一步操作是insert，那么在插入之前，第一个字符串的前 i 个字母已经与第二个字符串的前 j-1 个字母相等，因此，这种情况下的最小值是 f[i, j-1] + 1
    • delete：如果最后一步操作是delete，那么在删除之前，第一个字符串的前i-1个字母已经与第二个字符串的前 j 个字母相等，因此，这种情况下的最小值是 f[i-1, j] + 1
    • replace：替换分为两种情况，一、第一个字符串的前i个字母已经与第二个字符串的前j个字母相同，不需要进行替换操作，所以最小值是f[i-1,j-1]；二、第一个字符串的前i个字母与第二个字符串的前j个字母不同，但是第一个字符串的前 i-1个字母与第二个字符串的前 j-1 个字母相同，所以最小值是 f[i-1,j-1] + 1

class Solution {
    public int minDistance(String word1, String word2) {
        int n = word1.length();
        int m = word2.length();
        int[][] dp = new int[n + 1][m + 1];
        // 边界条件，将第一个字符串的前i个字母变为第二个字符串的前0个字母的操作数为i，逐步删除
        for (int i = 0; i <= n; i++) {
            dp[i][0] = i;
        }
        // 边界条件，将第一个字符串的前0个字母变为第二个字符串的前i个字母的操作数为i，逐步添加
        for (int i = 0; i <= m; i++) {
            dp[0][i] = i;
        }

        // 将第一个字符串的前i个字母变为第二个字符串的前j个字母
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                // 最后一步操作时insert或者delete
                dp[i][j] = Math.min(dp[i][j - 1], dp[i - 1][j]) + 1;
                // 最后一步操作时替换（可能替换，可能不替换）
                // 如果word1的第i个字母(word1[i-1])与word2的第j个字母(word2[j-1])不想等，则需要进行替换操作
                if (word1.charAt(i - 1) != word2.charAt(j - 1)) {
                    dp[i][j] = Math.min(dp[i][j], dp[i - 1][j - 1] + 1);
                } else {
                    //word1的第i个字母与word2的第j个字母相等，不需要进行替换操作
                    dp[i][j] = Math.min(dp[i][j], dp[i - 1][j - 1]);
                }
            }
        }
        return dp[n][m];
    }
}
