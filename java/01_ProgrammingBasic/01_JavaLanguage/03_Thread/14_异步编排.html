# 异步编排

# 一、启动线程的几种方法

  1. 继承Thread类

    class Thread01 extends Thread {
    @Override
    public void run() {
        System.out.println("Current Thread: " + Thread.currentThread().getName());
    }
}

    public class ThreadTest {
    public static void main(String[] args) {
        Thread01 thread01 = new Thread01();
        thread01.setName("extend Thread class");
        thread01.start();
    }
}

  2. 实现Runnable接口

    class Thread02 implements Runnable {
    @Override
    public void run() {
        System.out.println("Current Thread: " + Thread.currentThread().getName());
    }
}

    public class ThreadTest {
    public static void main(String[] args) {
        Thread02 thread02 = new Thread02();
        new Thread(thread02, "implements Runnable interface").start();
    }
}

  3. 实现Callable接口

    结合FutureTask，能获取返回值，可以处理异常

    class Thread03 implements Callable<String> {

    @Override
    public String call() throws Exception {
        System.out.println("Current Thread: " + Thread.currentThread().getName());
        return "Thread03 implements Callable interface";
    }
}

    public class ThreadTest {
    public static void main(String[] args) throws ExecutionException, InterruptedException {
        FutureTask<String> futureTask = new FutureTask<>(new Thread03());
        new Thread(futureTask, "implements Callable interface").start();
        // 等待线程执行完毕，获取返回结果，但是会阻塞线程
        String result = futureTask.get();
        System.out.println("result = " + result);
    }
}

  4. 线程池

    submit 执行任务 Runnable 任务

    Future<?> submit(Runnable task);

    public class ThreadTest {
    public static void main(String[] args) throws ExecutionException, InterruptedException {
        ExecutorService executorService = Executors.newFixedThreadPool(10);
        Future<?> future = executorService.submit(new Thread02());
        System.out.println(future.get());
    }
}

    submit 执行任务 Callable 任务

    <T> Future<T> submit(Callable<T> task);

    public class ThreadTest {
    public static void main(String[] args) throws ExecutionException, InterruptedException {
        ExecutorService executorService = Executors.newFixedThreadPool(10);
        Future<?> future = executorService.submit(new Thread03());
        System.out.println(future.get());
    }
}

    **总结：**submit 在执行 Callable任务时，能获取线程的返回结果

    execute 执行任务 Runnable 任务

    void execute(Runnable command);

    public class ThreadTest {
    public static void main(String[] args) {
        ExecutorService executorService = Executors.newFixedThreadPool(10);
        executorService.execute(new Thread02());
    }
}

# 二、CompletableFuture 异步编排

# 2.1 创建异步对象

CompletableFuture提供了以下方法来创建异步对象

  • 没有返回值的：

    public static CompletableFuture<Void> runAsync(Runnable runnable)
public static CompletableFuture<Void> runAsync(Runnable runnable, Executor executor)

    public class CompletableFutureDemo {
    public static void main(String[] args) throws ExecutionException, InterruptedException {
        CompletableFuture<Void> completableFuture = CompletableFuture.runAsync(() -> {
            System.out.println("runAsync");
        });

        ExecutorService threadPool = Executors.newFixedThreadPool(10);
        CompletableFuture<Void> completableFuture1 = CompletableFuture.runAsync(() -> {
            System.out.println("runAsync");
        }, threadPool);
    }
}

  • 有返回值的：

    public static <U> CompletableFuture<U> supplyAsync(Supplier<U> supplier)
    
public static <U> CompletableFuture<U> supplyAsync(Supplier<U> supplier, Executor executor)

    public class CompletableFutureDemo {
    public static void main(String[] args) {
        CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync");
            return "supplyAsync completableFuture";
        });
        System.out.println(completableFuture.join());

        ExecutorService threadPool = Executors.newFixedThreadPool(10);
        CompletableFuture<String> completableFuture1 = CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync with thread pool");
            return "supplyAsync completableFuture1";
        }, threadPool);
        System.out.println(completableFuture1.join());
    }
}

# 2.2 计算完成时回调方法

public CompletableFuture<T> whenComplete(BiConsumer<? super T, ? super Throwable> action)
    
public CompletableFuture<T> whenCompleteAsync(BiConsumer<? super T, ? super Throwable> action)
    
public CompletableFuture<T> whenCompleteAsync(BiConsumer<? super T, ? super Throwable> action, Executor executor)
    
public CompletableFuture<T> exceptionally(Function<Throwable, ? extends T> fn)

whenComplete可以处理正常和异常的计算结果，exceptionally处理异常情况。

whenComplete和whenCompleteAsync的区别：

  • whenComplete：是执行当前任务的线程，继续执行whenComplete的任务
  • whenCompleteAsync：把whenCompleteAsync中待执行的任务提交到线程池，由线程池分配线程来执行

总结：以Async结尾的方法，会将任务提交到线程池，由线程池分配线程来执行任务，可能与前一阶段执行任务的线程是同一个线程，也可能不同；不以Async结尾的方法，会由前一阶段执行任务的线程继续执行接下来的任务

exceptionally：相当于finally，一定会执行

public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync");
            int i = 10 / 0;
            return "first run result";
        }, threadPool).whenComplete((lastResult, exception) -> {
            System.out.println("the result is " + lastResult);
            System.out.println("the exception is " + exception);
        }).exceptionally(throwable -> {
            System.out.println("handle exception, the exception is " + throwable);
            return "default value";
        });
        System.out.println(completableFuture.join());
    }
}

# 2.3 handle 方法

能够处理返回值，也能够处理异常

public <U> CompletableFuture<U> handle(BiFunction<? super T, Throwable, ? extends U> fn)
    
public <U> CompletableFuture<U> handleAsync(BiFunction<? super T, Throwable, ? extends U> fn)
    
public <U> CompletableFuture<U> handleAsync(BiFunction<? super T, Throwable, ? extends U> fn, Executor executor)

public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
            System.out.println("supplyAsync");
            int i = 1 / 0;
            return "supplyAsync";
        }, threadPool).handle((lastResult, exception) -> {
            System.out.println("the last result is : " + lastResult);
            System.out.println("the exception is : " + exception);
            // 有结果，返回结果
            if (lastResult != null) {
                return lastResult;
            }
            // 有异常，返回异常
            if (exception != null) {
                return String.valueOf(exception);
            }
            // 没有结果，没有异常，返回默认值
            return "default value";
        });
        System.out.println("the final result is : " + completableFuture.join());
    }
}

# 2.4 线程串行化方法

在线程串行化中，如果中途某一个任务执行失败，则会终止执行剩下的任务

  1. thenApply

    获取上一个任务的执行结果，并返回当前任务的执行结果

    public <U> CompletableFuture<U> thenApply(Function<? super T, ? extends U> fn) 

public <U> CompletableFuture<U> thenApplyAsync(Function<? super T, ? extends U> fn) 

public <U> CompletableFuture<U> thenApplyAsync(Function<? super T, ? extends U> fn, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<Integer> completableFuture = CompletableFuture.supplyAsync(() -> {
            return ThreadLocalRandom.current().nextInt(10);
        }, threadPool).thenApply(lastResult -> {
            return lastResult + ThreadLocalRandom.current().nextInt(100);
        });
        System.out.println("the final result = " + completableFuture.join());
    }
}

  2. thenAccept

    获取上一个任务的执行结果，不返回当前任务的执行结果

    public CompletableFuture<Void> thenAccept(Consumer<? super T> action) 

public CompletableFuture<Void> thenAcceptAsync(Consumer<? super T> action) 

public CompletableFuture<Void> thenAcceptAsync(Consumer<? super T> action, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<Void> completableFuture = CompletableFuture.supplyAsync(() -> {
            return ThreadLocalRandom.current().nextInt(10);
        }, threadPool).thenAccept(lastResult -> {
            System.out.println("lastResult = " + lastResult);
        });
        System.out.println("the final result = " + completableFuture.join());
    }
}

  3. thenRun

    不获取上一个任务的执行结果，也不返回当前任务的执行结果

    public CompletableFuture<Void> thenRun(Runnable action) 

public CompletableFuture<Void> thenRunAsync(Runnable action) 

public CompletableFuture<Void> thenRunAsync(Runnable action, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<Void> completableFuture = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute first task");
            return ThreadLocalRandom.current().nextInt(10);
        }, threadPool).thenRun(() -> {
            System.out.println("execute second task");
        });
        System.out.println("the final result = " + completableFuture.join());
    }
}

# 2.5 两任务组合——都要完成

  1. thenCombine

    组合两个CompletableFuture，获取两个CompletableFuture任务的返回结果，并返回当前任务的执行结果

    public <U, V> CompletableFuture<V> thenCombine(CompletionStage<? extends U> other, BiFunction<? super T, ? super U, ? extends V> fn) 

public <U, V> CompletableFuture<V> thenCombineAsync(CompletionStage<? extends U> other, BiFunction<? super T, ? super U, ? extends V> fn)

public <U, V> CompletableFuture<V> thenCombineAsync(CompletionStage<? extends U> other, BiFunction<? super T, ? super U, ? extends V> fn, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute first task");
            return "first task";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task";
        }, threadPool);

        CompletableFuture<String> thirdTask = firstTask.thenCombine(secondTask, (firstResult, secondResult) -> {
            System.out.println("execute third task");
            System.out.println("firstResult = " + firstResult);
            System.out.println("secondResult = " + secondResult);
            return "third task";
        });
        
        System.out.println(thirdTask.join());
    }
}

  2. thenAcceptBoth

    组合两个CompletableFuture，获取两个CompletableFuture任务的返回结果，不会返回当前任务的执行结果

    public <U> CompletableFuture<Void> thenAcceptBoth(CompletionStage<? extends U> other, BiConsumer<? super T, ? super U> action) 

public <U> CompletableFuture<Void> thenAcceptBothAsync(CompletionStage<? extends U> other, BiConsumer<? super T, ? super U> action) 
    
public <U> CompletableFuture<Void> thenAcceptBothAsync(CompletionStage<? extends U> other, BiConsumer<? super T, ? super U> action, Executor executor)

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute first task");
            return "first task";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task";
        }, threadPool);

        CompletableFuture<Void> thirdTask = firstTask.thenAcceptBoth(secondTask, (firstResult, secondResult) -> {
            System.out.println("execute third task");
            System.out.println("firstResult = " + firstResult);
            System.out.println("secondResult = " + secondResult);
        });
        System.out.println(thirdTask.join());
    }
}

  3. runAfterBoth

    组合两个CompletableFuture，不会获取CompletableFuture任务的结果，也不会返回当前任务的执行结果

    public CompletableFuture<Void> runAfterBoth(CompletionStage<?> other, Runnable action) 

public CompletableFuture<Void> runAfterBothAsync(CompletionStage<?> other, Runnable action)

    public class CompletableFutureDemo {

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute first task");
            return "first task";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");

# 2.6 两任务组合——一个完成

  1. applyToEither

    两个任务中，有一个执行完毕，就获取其返回值，并返回当前任务的返回值

    public <U> CompletableFuture<U> applyToEither(CompletionStage<? extends T> other, Function<? super T, U> fn) 

public <U> CompletableFuture<U> applyToEitherAsync(CompletionStage<? extends T> other, Function<? super T, U> fn) 

public <U> CompletableFuture<U> applyToEitherAsync(CompletionStage<? extends T> other, Function<? super T, U> fn, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            System.out.println("execute first task");
            return "first task execute result";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task execute result";
        }, threadPool);

        CompletableFuture<String> thirdTask = firstTask.applyToEither(secondTask, (oneResult) -> {
            System.out.println(oneResult);
            return "third task execute result";
        });

        System.out.println(thirdTask.join());
    }
}

  2. acceptEither

    两个任务中，有一个执行完毕，就获取其返回值，不返回当前任务的返回值

    public CompletableFuture<Void> acceptEither(CompletionStage<? extends T> other, Consumer<? super T> action) 

public CompletableFuture<Void> acceptEitherAsync(CompletionStage<? extends T> other, Consumer<? super T> action) 

public CompletableFuture<Void> acceptEitherAsync(CompletionStage<? extends T> other, Consumer<? super T> action, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            System.out.println("execute first task");
            return "first task execute result";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task execute result";
        }, threadPool);

        CompletableFuture<Void> thirdTask = firstTask.acceptEither(secondTask, (oneResult) -> {
            System.out.println(oneResult);
        });

        System.out.println(thirdTask.join());
    }
}

  3. runAfterEither

    两个任务中，有一个执行完毕，就触发第三个任务，不获取前面任务的返回值，也不返回当前任务的执行结果

    public CompletableFuture<Void> runAfterEither(CompletionStage<?> other, Runnable action) 

public CompletableFuture<Void> runAfterEitherAsync(CompletionStage<?> other, Runnable action) 

public CompletableFuture<Void> runAfterEitherAsync(CompletionStage<?> other, Runnable action, Executor executor) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            System.out.println("execute first task");
            return "first task execute result";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task execute result";
        }, threadPool);

        CompletableFuture<Void> thirdTask = firstTask.runAfterEither(secondTask, () -> {
            System.out.println("execute third task");
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        });

        System.out.println(thirdTask.join());
    }
}

# 2.7 多任务组合

  1. allOf

    等待所有任务执行完成，才会触发当前任务

    public static CompletableFuture<Void> allOf(CompletableFuture<?>... cfs) 

    public class CompletableFutureDemo {
    static ExecutorService threadPool = Executors.newFixedThreadPool(10);

    public static void main(String[] args) {
        CompletableFuture<String> firstTask = CompletableFuture.supplyAsync(() -> {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            System.out.println("execute first task");
            return "first task execute result";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task execute result";
        }, threadPool);

        CompletableFuture<Void> completableFuture = CompletableFuture.allOf(firstTask, secondTask);
        System.out.println("completableFuture.join() = " + completableFuture.join());
        System.out.println("firstTask.join() = " + firstTask.join());
        System.out.println("secondTask.join() = " + secondTask.join());
    }
}

  2. anyOf

    只某一个任务执行完成，就会触发当前任务

    public static CompletableFuture<Object> anyOf(CompletableFuture<?>... cfs) 

    public class CompletableFutureDemo {

                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }

            return "first task execute result";
        }, threadPool);

        CompletableFuture<String> secondTask = CompletableFuture.supplyAsync(() -> {
            System.out.println("execute second task");
            return "second task execute result";

        System.out.println("secondTask.join() = " + secondTask.join());
    }
}
