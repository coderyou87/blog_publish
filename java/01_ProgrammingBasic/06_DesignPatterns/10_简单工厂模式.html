# 简单工厂模式

# 一、介绍

三个角色

Factory：工厂角色， CarFactory

Product：抽象产品角色，Car

ConcreteProduct：具体产品角色， VanCar、MiniCar

缺点：违背开闭原则，不易扩展

# 二、实现

public abstract class Car {

    protected String engine;

    public abstract void run();
}

public class VanCar extends Car {

    @Override
    public void run() {
        System.out.println("VarCar run, engine is " + engine);
    }
}

public class MiniCar extends Car {

    @Override
    public void run() {
        System.out.println("MiniCar run, engine is " + engine);
    }
}

简单工厂实现：

public class CarFactory {
    public Car newCar(String type) {
        Car car = null;
        if ("VanCar".equals(type)) {
            car = new VanCar();
            car.engine = "VanCar Engine";
        } else if ("MiniCar".equals(type)) {
            car = new MiniCar();
            car.engine = "MiniCar Engine";
        }

        return car;
    }
}

public class SimpleFactoryTest {
    public static void main(String[] args) {
        CarFactory carFactory = new CarFactory();
        Car vanCar = carFactory.newCar("VanCar");
        vanCar.run();

        Car miniCar = carFactory.newCar("MiniCar");
        miniCar.run();
    }
}
