# 1.1 Resource

定义：

package org.springframework.core.io;
public interface Resource extends InputStreamSource

Resource是各种资源的顶级接口，能够表示各种资源。最常见的是，ResourceLoader将bean.xml文件加载到内存中，并将其解析转换为Resource。

主要实现类有：

  • HttpResource：主要是从网络中获取资源

  • ContextResource：从类路径下获取资源

  • WritableResource：从文件中获取资源

# 1.3 BeanFactory

定义信息：

/**
 * BeanFactory是访问Spring Bean 容器的根接口
 * 在这里，保存了一些Bean的定义信息，每一个Bean的定义信息，都有一个唯一的名称
 * 工厂将会根据这些Bean定义信息，生成多实例Bean（原型模式）或者单实例Bean（单例模式）
 * 使用的是工厂方法模式
 */
package org.springframework.beans.factory;
public interface BeanFactory

主要方法：

主要继承接口与实现类：

  • HierarchicalBeanFactory：定义父子工厂（父子容器）

  • ListableBeanFactory：ListableBeanFactory的一个实现类是DefaultListableBeanFactory，在DefaultListableBeanFactory类中，保存了 IOC 容器的核心信息

  • AutowireCapableBeanFactory：提供自动装配能力

虽然AnnotationConfigWebApplicationContext没有自动装配能力，但是DefaultListableBeanFactory有自动装配能力。
