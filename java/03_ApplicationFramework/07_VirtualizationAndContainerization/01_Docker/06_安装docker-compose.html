# 安装docker-compose

  1. 下载

    从GitHub下载docker-compose文件

    curl -L https://github.com/docker/compose/releases/download/1.26.2/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose-back

    如果GitHub下载速度较慢，可以使用一下连接下载

    curl -L https://get.daocloud.io/docker/compose/releases/download/1.26.2/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose

  2. 赋权

    chmod +x /usr/local/bin/docker-compose

  3. 查看版本信息

    docker-compose -v

  4. 卸载

    rm -rf /usr/local/bin/docker-compose
