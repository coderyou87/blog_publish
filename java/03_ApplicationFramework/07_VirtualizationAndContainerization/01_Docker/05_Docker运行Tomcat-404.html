# Docker运行Tomcat_404

  1. 安装Tomcat

    docker pull tomcat:8.5

  2. 运行Tomcat

    docker run -d -p 80:8080 tomcat:8.5

  3. 访问Tomcat

    http://192.168.76.3

  4. 排查、解决问题

    是否开放80端口

    firewall-cmd --list-ports

    以交互模式进入Tomcat安装目录

    docker exec -it containerId /bin/bash

    查看webapps目录

    ls ./webapps

    将webapps.dist目录下的内容复制到webapps目录下

    cp -r ./webapps.dist/* ./webapps

  5. 再次访问
