# MyBatis动态SQL

# 一、if

一般用在查询的where子句中，如果有某个条件，就携带上这个条件。

  1. 实现一

    • 接口

      void insertAttr(@Param("pmsAttr") PmsAttr pmsAttr);

    • xml文件

      <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    where 1 = 1
    <if test="pmsAttr != null and pmsAttr.attrId != null">
        and attr_id = #{pmsAttr.attrId}
    </if>
    <if test="pmsAttr != null  and pmsAttr.attrName != null">
        and attr_name = #{pmsAttr.attrName}
    </if>
</select>

# 二、where

where标签会去掉第一个and`

  • 接口

    void insertAttr(@Param("pmsAttr") PmsAttr pmsAttr);

  • xml文件

    <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    <where>
        <if test="pmsAttr != null and pmsAttr.attrId != null">
            and attr_id = #{pmsAttr.attrId}
        </if>
        <if test="pmsAttr != null  and pmsAttr.attrName != null">
            and attr_name = #{pmsAttr.attrName}
        </if>
    </where>
</select>

# 三、choose (when, otherwise)

相当于switch--case，每一次只会进入一个分支。如果没有进入when分支，则会进入otherwise分支。如果满足多个when分支，则会进入第一个when分支。

  • 接口

    void insertAttr(@Param("pmsAttr") PmsAttr pmsAttr);

  • xml文件

    <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    <where>
        <choose>
            <when test="pmsAttr != null and pmsAttr.attrId != null">
                attr_id = #{pmsAttr.attrId}
            </when>
            <when test="pmsAttr != null and pmsAttr.attrName != null">
                attr_name = #{pmsAttr.attrName}
            </when>
            <otherwise>
                1 = 1
            </otherwise>
        </choose>
    </where>
</select>

# 四、trim

trim的参数解释：

  1. prefix：给拼串后的字符串添加一个前缀

  2. prefixOverrides：去除整个字符串前面多余的前缀

  3. suffix：给拼串后的字符串添加一个后缀

  4. suffixOverrides：去除整个字符串后面多余的后缀

  5. 示例一：

    • 接口

      List<PmsAttr> queryAttr(@Param("pmsAttr") PmsAttr pmsAttr);

    • xml文件

      <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    <trim prefix="where" prefixOverrides="and">
        <if test="pmsAttr != null and pmsAttr.attrId != null">
            and attr_id = #{pmsAttr.attrId}
        </if>
        <if test="pmsAttr != null  and pmsAttr.attrName != null">
            and attr_name = #{pmsAttr.attrName}
        </if>
    </trim>
</select>

  6. 示例二：

    <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    <trim prefix="where" suffixOverrides="and">
        <if test="pmsAttr != null and pmsAttr.attrId != null">
             attr_id = #{pmsAttr.attrId} and
        </if>
        <if test="pmsAttr != null  and pmsAttr.attrName != null">
             attr_name = #{pmsAttr.attrName} and
        </if>
    </trim>
</select>

# 五、set

  1. 原始

    • 接口

      void updateAttr(PmsAttr pmsAttr);

    • xml文件

      <update id="updateAttr">
    update pms_attr
    set
    <if test="attrName != null">
        attr_name = #{attrName},
    </if>
    <if test="icon != null">
        icon = #{icon},
    </if>
    where attr_id = #{attrId}
</update>

    • 问题：如果attrName不为空，而icon为空，则会多出attrName后面的逗号

  2. 使用set标签

    • 接口

      void updateAttr(PmsAttr pmsAttr);

    • xml文件

      <update id="updateAttr">
    update pms_attr
    <set>
        <if test="attrName != null">
            attr_name = #{attrName},
        </if>
        <if test="icon != null">
            icon = #{icon},
        </if>
    </set>
    <where>
        attr_id = #{attrId}
    </where>
</update>

  3. 使用trim标签

    • 接口

      void updateAttr(PmsAttr pmsAttr);

    • xml文件

      <update id="updateAttr">
    update pms_attr
    <trim prefix="set" suffixOverrides=",">
        <if test="attrName != null">
            attr_name = #{attrName},
        </if>
        <if test="icon != null">
            icon = #{icon},
        </if>
    </trim>
    <where>
        attr_id = #{attrId}
    </where>
</update>

# 六、foreach

foreach中的参数说明：

  • collection：指定要遍历的集合
  • item：遍历出的每一个元素，会赋值给item
  • open：以open开始
  • close：以close结束
  • separator：分隔符
  • index：遍历list时是索引，遍历map时是index表示map中的key，item表示map中的值

# 6.1查询

  • 接口

    List<PmsAttr> queryAttr(List<String> ids);

  • xml文件

    <select id="queryAttr" resultType="pmsAttr">
    select *
    from pms_attr
    <where>
        attr_id in
        <foreach collection="list" item="id" open="(" close=")" separator=",">
            #{id}
        </foreach>
    </where>
</select>

# 七、bind

  1. 模糊查询

    • 接口

      List<PmsAttr> queryAttr(String name);

    • xml文件

      <select id="queryAttr" resultType="pmsAttr">
    <bind name="pattern" value="'%' + name + '%'"/>
    select *
    from pms_attr
    <where>
        attr_name like #{pattern}
    </where>
</select>

    • sql

      select * from pms_attr WHERE attr_name like ?

# 八、sql

    category_id, show_desc
</sql>

  2. 在其他地方引用

    在引用时，仍然可以通过property标签添加其他列

    新增

        <include refid="attr"></include>

# 九、MyBatis中两个重要参数

  1. _databaseId：数据库的Id
  2. _parameter：传入的参数。如果传入的只有一个参数，_parameter就代表当前参数；如果传入的是一个Map，就代表这个Map
