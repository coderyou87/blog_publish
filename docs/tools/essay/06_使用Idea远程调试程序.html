# 使用Idea远程调试程序

  1. 将程序上传到服务器

  2. 启动服务器上的程序

    java -Xdebug -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=8089 -jar testremote-0.0.1-SNAPSHOT.jar

    address：监听的端口号

  3. 配置Idea中的remote

    JVM参数：

    -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=8089

  4. 启动TestRemote

  5. 调试
