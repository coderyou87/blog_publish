# Docker部署Vue项目

  1. 在docker中安装nginx

    docker pull nginx

  2. 将前端项目打包并上传至服务器

    将前端打包的文件夹上传至/data/vue，

    /data/vue/dist
css  favicon.ico  img  index.html  js

  3. 准备nginx配置文件

    /data/vue/nginx.conf

    编辑nginx.conf文件

    server {
	listen 82;
	server_name 192.168.76.3;
	location / {
		root /data/vue/dist;
		index index.html index.htm;
	}
}

  4. 开放端口

    firewall-cmd --zone=public --add-port=82/tcp --permanent
firewall-cmd --reload

  5. 在docker中运行nginx

    docker run -d -p 82:82 --name nginx \
-v /data/vue:/data/vue \
-v /data/vue/nginx.conf:/etc/nginx/nginx.conf \
nginx

  6. 访问页面

    http://192.168.76.3:82
