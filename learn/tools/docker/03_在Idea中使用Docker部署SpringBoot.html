# 在 Idea 中使用 Docker 部署 SpringBoot.md

  1. 修改 Docker 服务文件

    vim /lib/systemd/system/docker.service

    配置 tcp 监听端口并开放端口

    ExecStart=/usr/bin/dockerd -H tcp://0.0.0.0:2375 -H unix:///var/run/docker.sock

    开放端口

    firewall-cmd --zone-public --add-port=2375/tcp --premanent
firewall-cmd --reload

  2. 使用 Idea 创建一个 SpringBoot web 应用，并保证能够在本地运行成功

  3. 在项目中集成 Docker 插件

    修改 pom.xml 文件

    添加 Docker 镜像的前缀

    <properties>
    <java.version>1.8</java.version>
    <docker.image.prefix>docker</docker.image.prefix>
</properties>

    添加一个 plugin

    <plugin>
    <groupId>com.spotify</groupId>
    <artifactId>docker-maven-plugin</artifactId>
    <version>1.0.0</version>
    <configuration>
        <!--Docker远程地址（Docker的安装地址）-->
        <dockerHost>http://192.168.76.3:2375</dockerHost>
        <!--镜像名称（前缀/项目名）-->
        <imageName>${docker.image.prefix}/${project.artifactId}</imageName>
        <!--Dockerfile文件的位置-->
        <dockerDirectory>src/main/docker</dockerDirectory>
        <resources>
            <resource>
                <targetPath>/</targetPath>
                <directory>${project.build.directory}</directory>
                <include>${project.build.finalName}.jar</include>
            </resource>
        </resources>
    </configuration>
</plugin>

  4. 编写 dockerfile 文件

    在src/main/目录下创建一个文件夹，命名为 docker

    在src/main/docker文件夹下创建一个文件，命名为 Dockerfile，内容为：

VOLUME /tmp
ADD testdocker-0.0.1-SNAPSHOT.jar /testdocker.jar
ENTRYPOINT ["java", "-Djava.security.egd=file:/dev/./urandom", "-jar", "/testdocker.jar"]
