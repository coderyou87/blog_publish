# 在CentOS7中，将MySQL注册为系统服务

  1. 确保MySQL安装成功并能够运行

    能够以$MYSQL_HOME/support-files/mysql.server脚本运行mysql，在运行之前，需要修改mysql.server中的basedir与datadir。

    basedir=/opt/module/mysql-5.7.25-el7-x86_64
datadir=/opt/module/mysql-5.7.25-el7-x86_64/data

  2. 编写系统服务运行脚本

    vim /usr/lib/systemd/system/mysql.service

    # mysql.service

[Unit]
Description=mysql service
After=mysql.service

[Service]
Type=forking
User=users
Group=users
PIDFile=/opt/module/mysql-5.7.25-el7-x86_64/data.localhost.localdomain.pid
ExecStart=/opt/module/mysql-5.7.25-el7-x86_64/support-files/mysql.server start
ExecReload=
ExecStop=/opt/module/mysql-5.7.25-el7-x86_64/support-files/mysql.server stop
PrivateTmp=true
[Install]
    1. [Unit]：对当前服务进行说明
      • Descriptor：描述服务
      • After：描述服务类别
    2. [Service]：服务的关键部分，设置服务的运行参数等，服务的启动与停止等命令，都需要使用绝对路径。
      • Type=forking：后台运行服务
      • User=users：设置服务运行的用户
      • Group=users：设置服务运行的用户组
      • PIDFile：存放服务PID的文件路径
      • ExecStart：启动服务的命令
      • ExecReload：重启服务的命令
      • ExecStop：停止服务的命令
      • PrivateTmp=true：微服务分配单独的临时空间
    3. [Install]：服务安装的相关设置

  3. 在Centos7中，服务相关的命令有：

    1. 启动服务：systemctl start XXX
    2. 停止服务：systemctl stop XXX
    3. 查看服务状态：systemctl status XXX
