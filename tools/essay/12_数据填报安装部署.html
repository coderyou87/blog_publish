# 数据填报安装部署

# 一、部署的内容

  1. 注册中心 nacos

    nacos管理所有服务地址、中转以及服务的配置信息

  2. 网关 gateway

    安全拦截与请求转发

  3. 各个微服务

    例如：数据填报、ue、dac

  4. 前端服务器nginx

# 二、部署详细步骤

# 2.1 安装注册中心

  1. 解压nacos

  2. 配置数据库

    • 安装数据库，版本要求：5.6.5+
    • 初始化mysql数据库，数据库初始化文件：nacos-mysql.sql
    • 修改conf/application.properties文件，增加支持mysql数据源配置（目前只支持mysql），添加mysql数据源的url、用户名和密码。
    spring.datasource.platform=mysql

db.num=1
db.url.0=jdbc:mysql://11.162.196.16:3306/nacos_devtest?characterEncoding=utf8&connectTimeout=1000&socketTimeout=3000&autoReconnect=true
db.user=nacos_devtest
db.password=youdontknow

  3. 启动nacos

    单机启动模式

    ${nacos-home}/bin/startup.sh -m standalone

  4. 查看管理界面

    访问：http://ip:8848/nacos/index.html

    初始用户名与密码都是nacos

# 2.2 启动微服务

  1. 每一个配置文件中，使用域名，不实用ip地址

  2. 在/etc/hosts中，添加域名配置

  3. 提供默认配置文件，在注册中心导入配置文件，并在注册中心修改修改配置文件，发布，即可。

  4. 启动微服务

    ${server-home}/bin/${server-name} start

    当前启动的服务主要有：df-editor、df-datasource、gateway

# 2.3 启动前端

  1. 安装Nginx

    • 解压nginx tar -zxvf nginx.tar.gz
    • 编译nginx ./configure --prefix=/xx/xx
    • 安装 make && make install
    • 修改nginx配置，更改前端页面路径(root)，修改网关地址

  2. 将前端项目解压在指定目录下

  3. 启动nginx

  4. 访问nginx
