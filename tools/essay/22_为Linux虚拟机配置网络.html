# 为Linux虚拟机配置网络

示例：

TYPE=Ethernet
PROXY_METHOD=none
BROWSER_ONLY=no
BOOTPROTO=static
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_FAILURE_FATAL=no
IPV6_ADDR_GEN_MODE=stable-privacy
NAME=ens33
UUID=729bcd54-b893-4e5b-961a-d539f3e9e3bb
DEVICE=ens33
ONBOOT=yes
MM_CONTROLLED=no
IPADDR=192.168.76.5
NETMASK=255.255.255.0
GATEWAY=192.168.76.2
DNS1=114.114.114.114
DNS2=8.8.8.8

在安装完虚拟机后，需要修改的有：

BOOTPROTO=static
ONBOOT=yes

需要添加的有：

IPADDR=192.168.76.5
NETMASK=255.255.255.0
GATEWAY=192.168.76.2
DNS1=114.114.114.114
DNS2=8.8.8.8

修改完成后，保存退出，更新网络（以下两行命令选其一）：

systemctl restart network
service network restart

查看IP信息：

ip addr
ifconfig

ifconfig需要安装net-tools后，才能使用
