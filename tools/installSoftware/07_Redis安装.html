# Redis安装与配置

# 安装

  1. 安装相关依赖

    yum -y install gcc automake autoconf libtool make

  2. 下载

    wget http://download.redis.io/releases/redis-5.0.8.tar.gz

  3. 解压

    tar -zxvf /opt/software/redis-5.0.8.tar.gz -C /opt/module/

  4. 编译

    cd /opt/module/redis-5.0.8/
make MALLOC=libc

  5. 安装

    make PREFIX=/opt/module/redis-5.0.8 install

  6. 启动

    /opt/module/redis-5.0.8/bin/redis-server /opt/module/redis-5.0.8/redis.conf
/opt/module/redis-5.0.8/bin/redis-cli

# 配置

  1. 注释掉bind，让所有的IP都可以访问

    bind 127.0.0.1

  2. 关闭保护模式

    protected-mode no

  3. 修改tcp-keepalive

    tcp-keepalive 30

  4. 开启守护进程

    daemonize yes

  5. 设置访问密码

    requirepass password**
