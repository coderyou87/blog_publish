# 安装Nacos

  1. 在数据库中新建一个数据库

    drop database if exists nacos_config;
create database if not exists nacos_config default character set utf8 collate utf8_general_ci;

  2. 在GitHub上下载nacos，解压到执行目录

    nacos下载地址，将文件放在/opt/software/目录下

    解压文件：

    tar -zxvf /opt/software/nacos-server-1.2.1.tar.gz -C /opt/module/

  3. 执行sql文件

    mysql -u root -p 
use nacos_config
source /opt/module/nacos/conf/nacos-mysql.sql

  4. 修改配置文件

    spring.datasource.platform=mysql
db.num=1
db.url.0=jdbc:mysql://ip:3306/nacos_config?characterEncoding=utf8&connectTimeout=1000&socketTimeout=3000&autoReconnect    =true
db.user=root
db.password=hanguang

  5. 启动

    /opt/module/nacos/bin/start.sh -m standalone

  6. 开放端口

    firewall-cmd --zone=public --add-port=8848/tcp --permanent
firewall-cmd --reload

  7. 验证

    http:ip:8848/nacos/index.html
nacos nacos
